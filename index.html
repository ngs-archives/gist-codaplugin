Gist Plug-In for Coda 2

How does it work

This plug-in allows you to create public and private gists from current active document.

The latest release

 

Shortcuts

^⌥⌘G
Create Private Gist
^⌥⇧⌘G
Create Public Gist
^⌥⌘S
Create Private Gist from selection
^⌥⇧⌘S
Create Public Gist from selection

Intalling

Download, Unzip and Double click, that's all.
